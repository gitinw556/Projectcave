## เครปสไตล์เรา

ขั้นตอนการสั่งเครป

1.เลือกแป้ง

2.เลือกแยม

3.เลือกไส้คาว

4.เลือกไส้หวาน

5.เลือกซอส

6.กรอกชื่อผู้สั่ง

7.กดปุ่ม OK เพื่อสั่งรายการที่ต้องการ

8.ถ้าเราต้องการจะเลือกใหม่ให้ กดปุ่ม Clear 

9.ถ้าเราไม่ต้องการสั่งเครปให้ กดปุ่ม Exit ได้เลยครับ

ราคารวมทั้งหมดก็จะโชว์ขึ้นตรง "ราคารวม"
